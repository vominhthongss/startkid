1. Chạy lệnh
   npm install
   hoặc
   yarn install
2. Chạy app
   \*Đối với ios
   npm run ios

   \*Đối với android
   npm run androind

3. Nếu dùng server tạm (mock data) thì (phải mở 1 terminal riêng)
   cd server (lúc này bạn đang ở /server)
   chạy lệnh
   json-server --watch db.json --port 3000
